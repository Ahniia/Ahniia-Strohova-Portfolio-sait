Restaurant

Overview

This site was my final project in my programming course. My assignment was to make a small restaurant website that would have a menu for all the dishes. Some dishes would be unavailable because they were out of stock. It was also necessary to make a button for a detailed description of each dish, and the ability to order this dish directly on the details page. A dish with a picture, short description and price should have been saved in the cart. Also in the cart should be calculated the total amount of the order. And if this order total exceeds 40 euros, a discount should have been applied.

My Role

Sole developer and designer

Duration

3 days

Tools

Angular, TypeScript, CSS, HTML, Visual Studio Code, Figma

Dish cards are created from an array of information in JSON format.
This array contains information about the name of the dish, photo, short description, price, and whether the dish is available.
Each card has a "details" button that leads to a page with that dish. And the button "order" which adds the dish to the cart.

This page, is also created from an array of information in JSON format. The card is created using Bootstrap. And the code for displaying the dish data in the card was written in the TypeScript. The dish page contains a photo of the dish, as well as a short description and price. In addition, there is a "buy" button that adds this dish to your cart.

The shopping cart displays the added dishes. Calculates the total cost of the ordered dishes, adds a 10% service charge, and then displays the total cost of the order.

In addition, if the total order amount exceeds 40 euros, then a 15% discount is displayed in the shopping cart. And the total cost of the order is reduced by 15%.
